How to define a table when using the following elements?

tr: Defines a row in a table
th: Defines a header cell in a table
td: Defines a cell in a table
thead: Groups the header content in a table
tfoot: Groups the footer content in a table
caption: Defines a table caption

What is the difference between cells spacing and padding?

Cellpadding defines the space present between a table cell's border and the content present in it. Cellspacing defines the space present between individual adjacent cells.

How do you define table borders using CSS?

To specify table borders in CSS, use the border property

Ex:table, th, td { border: 1px solid; }

How do you control border widths using CSS?

Using the border-width property, you can set the width of an element's four borders [div border-width: thin;}
